# Smmplan — Профессиональная SMM-панель

Проект по автоматизации продвижения в социальных сетях (Telegram, VK, YouTube, Twitch и др.). 
Базируется на Next.js 16, Tailwind 4 и Prisma 5.

## 🏁 С чего начать?

Вся база знаний по проекту собрана в папке **`docs`**. 

👉 **[Перейти к Инструкциям и Плану запуска (INDEX.md)](file:///d:/Smmplan/docs/INDEX.md)**

---

## ⚡ Быстрый запуск (Локально)

Для тех, кто уже знаком с проектом:
```bash
docker compose up -d db redis
npm install
npx prisma migrate dev
npm run dev
```
