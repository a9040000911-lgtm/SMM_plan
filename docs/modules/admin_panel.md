# Module: Admin Panel & Engines

## 📍 Назначение
Управление бизнес-логикой, ценами, провайдерами и обработка инцидентов.

## 🛠 Технический стек
- **Framework**: Next.js 15 (App Router)
- **UI**: Tailwind CSS + Lucide
- **Logic**: Server Actions

## 📋 Функционал
- **Markup Engine**: Система массового изменения цен с поддержкой % наценки, фиксированной суммы и Floor Price (минимальный порог).
- **Auto-Repricing (SmartSync)**: Фоновый процесс, синхронизирующий ваши цены с актуальными ценами провайдеров каждые 30-60 минут.
- **Support Dashboard (Command Center)**: Хаб реального времени с метриками заказов, платежей и инцидентов.
- **Service Health Monitor**: Аналитическая панель, оценивающая стабильность (Success Rate) и прибыльность каждой услуги на основе последних 500 транзакций.
- **Incident Radar**: Автоматическое выявление "зависших" заказов (в статусе Processing > 2 часов) для оперативного вмешательства саппорта.
- **Import Pipeline (AI-Ready)**: Массовый импорт услуг от провайдеров с автоматическим определением типов целей.
- **Order Pipeline**: Таблица заказов с группировкой по датам, мгновенной синхронизацией (Inline Sync) и безопасным возвратом (Refund).
- **Provider Management**: Полный CRUD провайдеров с мониторингом баланса.
