# Module: Telegram Bot

## 📍 Назначение
Основной интерфейс взаимодействия с клиентами. Обеспечивает выбор услуг, валидацию данных и оформление заказов.

## 🛠 Технический стек
- **Библиотека**: Telegraf.js
- **Сессии**: Redis (SessionService)
- **Файл**: `src/bot/index.ts`

## 📋 Функционал
- **Wizard Scenes**: Пошаговая логика (WizardScene).
    - `OrderWizard`: Оформление заказа (Ссылка -> Категория -> Тариф -> Кол-во).
    - `DepositWizard`: Пополнение баланса (Сумма -> Платежная ссылка).
    - `SupportWizard`: Создание тикета (Тема -> Описание + Фото).
- **Smart Pivot (Умный переключатель)**: Если пользователь присылает ссылку, не подходящую к выбранной услуге, бот автоматически пересчитывает категории и предлагает подходящие варианты.
- **Real-time Notifications**: Мгновенные уведомления о пополнении баланса, ответах саппорта и системных новостях через `BroadcastService`.
- **Double-Check Validation**: Ссылка проверяется дважды — при вводе и при выборе конкретного тарифа.
- **Интеграция с LinkService**: Использование семантического анализа ссылок для фильтрации каталога.

## 🏗 Структура
- `src/bot/index.ts`: Ядро, регистрация middleware и Stage.
- `src/bot/scenes/`: Папка с логикой пошаговых мастеров.
- `src/services/broadcast.service.ts`: Центр уведомлений (Админ/Юзер/Рассылки).
