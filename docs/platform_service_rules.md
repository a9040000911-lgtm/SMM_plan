# Спецификация типов ссылок и сопоставления услуг Smmplan

Этот документ является эталонным для работы `Smart Analyzer` и фильтрации услуг в каталоге.

## 1. Платформа: TELEGRAM

| Тип объекта | Идентификатор | Признак ссылки | Услуги |
| :--- | :--- | :--- | :--- |
| Канал / Группа | `TG_CHANNEL` | Корень (`t.me/name`) или инвайт (`t.me/+`) | Подписчики, Бусты, Пакеты просмотров (Last X), Автопросмотры (Future), Звезды на канал |
| Пост | `TG_POST` | Наличие ID в конце (`t.me/name/123`) | Просмотры (1 пост), Реакции, Репосты, Комментарии, Звезды на пост, Голоса в опрос |
| История | `TG_STORY` | Наличие `/s/` или `/stories/` | Просмотры историй, Реакции на истории, Репосты историй |
| Бот | `TG_BOT` | Наличие `?start=` | Рефералы, Запуски |

## 2. Платформа: VK (ВКонтакте)

| Тип объекта | Идентификатор | Признак ссылки | Услуги |
| :--- | :--- | :--- | :--- |
| Группа / Паблик | `VK_GROUP` | `public`, `club`, `event` или короткое имя (не `id`) | Подписчики в сообщество |
| Профиль | `VK_PROFILE` | `id123` или имя пользователя | Друзья, Подписчики на профиль |
| Пост на стене | `VK_WALL` | `wall-` | Лайки, Репосты, Просмотры, Комментарии |
| Видео | `VK_VIDEO` | `video-` | Просмотры видео, Лайки |
| Клип | `VK_CLIP` | `clip-` | Просмотры клипов, Лайки |
| Плейлист | `VK_PLAYLIST` | `music/playlist` или `audio_playlist` | Прослушивания |

## 3. Платформа: INSTAGRAM

| Тип объекта | Идентификатор | Признак ссылки | Услуги |
| :--- | :--- | :--- | :--- |
| Профиль | `IG_PROFILE` | Корень (`instagram.com/name`) | Подписчики |
| Пост | `IG_POST` | `/p/` или `/tv/` | Лайки, Просмотры, Комментарии, Сохранения |
| Reels | `IG_REEL` | `/reels/` | Просмотры Reels, Лайки |
| История | `IG_STORY` | `/stories/` | Просмотры историй |

## 4. Платформа: YOUTUBE

| Тип объекта | Идентификатор | Признак ссылки | Услуги |
| :--- | :--- | :--- | :--- |
| Канал | `YT_CHANNEL` | `@name` или `channel/` | Подписчики |
| Видео | `YT_VIDEO` | `watch?v=` или `youtu.be/` | Просмотры, Лайки, Комментарии, Часы просмотра |
| Shorts | `YT_SHORT` | `/shorts/` | Просмотры Shorts, Лайки |

*Документ утвержден AI-директором. Любые изменения в анализаторе должны соответствовать этой таблице.*
