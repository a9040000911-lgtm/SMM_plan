# Модуль: Smart Link Analyzer (Умный фильтр)

## 🧠 Принцип работы
Система перехватывает ввод ссылки пользователем и прогоняет её через набор регулярных выражений (Regex). 

### 1. Этап: Определение платформы и типа контента
| Тип ссылки | Платформа | Объект | Предлагаемые категории |
| :--- | :--- | :--- | :--- |
| `t.me/joinchat/...` | Telegram | Канал/Группа | Подписчики |
| `t.me/name` | Telegram | Канал/Группа | Подписчики, Просмотры (на последние посты) |
| `t.me/name/123` | Telegram | Пост | Просмотры, Реакции, Репосты |
| `vk.com/id123` | VK | Профиль | Друзья, Подписчики |
| `vk.com/wall-123_456` | VK | Пост | Лайки, Репосты, Просмотры |
| `instagram.com/p/...` | Instagram | Пост/Reels | Лайки, Просмотры, Комментарии, Сохранения |

### 2. Этап: Фильтрация по ГЕО
После определения категорий, пользователю предлагается переключатель ГЕО:
- **РФ / СНГ** (по умолчанию для локального рынка).
- **Весь мир** (Global).
- **Другие страны** (выпадающий список, скрыт под "спойлер").

### 3. Этап: Маппинг на Тарифы
В выбранной категории (например, "Telegram Подписчики") система ищет активные внутренние тарифы.
