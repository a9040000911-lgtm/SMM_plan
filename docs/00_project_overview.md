# Smmplan: Проект инновационной SMM-панели

## 🎯 Общее видение
Smmplan — это современная платформа для перепродажи услуг продвижения в социальных сетях (лайки, подписчики, репосты и др.). Проект ориентирован как на розничных клиентов, так и на оптовых реселлеров.

## 🚀 Цели MVP
1. **Интеграция с провайдером**: Полноценная работа с API VexBoost.
2. **Каталог услуг**: Автоматический импорт и обновление услуг.
3. **Система заказов**: Создание, отслеживание и обработка заказов.
4. **Личный кабинет**: Регистрация, авторизация и пополнение баланса.
5. **Админ-панель**: Управление наценками, пользователями и заказами.

## 🔑 Ключевые особенности
- Динамическое ценообразование (наценки для разных групп пользователей).
- Высокая скорость обработки заказов через API.
- Современный, интуитивно понятный UI/UX.
