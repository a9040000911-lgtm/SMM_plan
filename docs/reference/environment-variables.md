# 📖 Справочник: Переменные окружения (.env)

Этот справочник описывает все доступные переменные окружения, которые используются в **Smmplan**.

> [!WARNING]
> Никогда не коммитьте реальные ключи авторизации и токены в Git.

## Базовые системные настройки (System Settings)

| Переменная | Описание | Значение по умолчанию | Обязательность |
|------------|----------|-----------------------|----------------|
| `NODE_ENV` | Режим работы (`development`, `production`, `test`). | `development` | Да |
| `PORT`     | Порт, на котором стартует Next.js-сервер. | `3000` | Нет |

## База Данных (Database)

| Переменная | Описание | Пример | Обязательность |
|------------|----------|--------|----------------|
| `DATABASE_URL` | Строка подключения к PostgreSQL (формат Prisma). | `postgresql://user:pass@host:5432/db` | Да |

## Авторизация и JWT (Auth & Security)

| Переменная | Описание | Пример | Обязательность |
|------------|----------|--------|----------------|
| `JWT_SECRET_KEY` | Ключ для подписи JWT-токенов. Сгенерируйте `openssl rand -hex 32` | `...` | Да |
| `NEXTAUTH_URL` | Канонический URL сайта для колбеков (SSO). | `http://localhost:3000` | Да |

## Telegram Интеграция (Telegram Bot API)

| Переменная | Описание | Пример | Обязательность |
|------------|----------|--------|----------------|
| `TELEGRAM_BOT_TOKEN` | Токен бота от `@BotFather`. | `1234:ABCDEFG` | Да |
| `TELEGRAM_WEBHOOK_URL` | URL для получения вебхуков из TG (для продакшена). | `https://smmplan.ru/api/webhook` | Зависит |

## Платежные Шлюзы (Payment Gateways)

| Переменная | Описание | Пример | Обязательность |
|------------|----------|--------|----------------|
| `YOOKASSA_SHOP_ID` | Идентификатор магазина YooKassa (B2B/B2C RF). | `123456` | Нет |
| `YOOKASSA_SECRET_KEY`| Секретный ключ YooKassa. | `live_...` | Нет |
| `CRYPTOPAY_TOKEN` | Токен Crypto Bot для оплаты криптовалютой. | `...` | Нет |

## Различные API (Misc)

| Переменная | Описание | Пример | Обязательность |
|------------|----------|--------|----------------|
| `CBR_API_URL` | URL API Центробанка РФ для курсов валют (XML). | `http://www.cbr.ru/scripts/XML_daily.asp` | Нет |
