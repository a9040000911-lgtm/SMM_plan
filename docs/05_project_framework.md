# Каркас проекта Smmplan

## 1. Модуль "Умный Фильтр" (Smart Link Analyzer)
**Бизнес-логика**: Когда пользователь вставляет ссылку, система анализирует её и:
- Определяет соцсеть (Instagram, Telegram, VK и т.д.).
- Определяет тип контента (Пост, Канал, Группа, Reels, Story).
- **Результат**: Пользователю показывают только те категории и услуги, которые применимы к этой ссылке.

## 2. Модуль "Тарификация и Ранжирование"
**Бизнес-логика**: Вместо сотен дублирующихся услуг от провайдера мы создаем внутренние "Тарифы":
- **Структура**: Соцсеть -> Категория (Подписчики) -> Тариф (Эконом, Стандарт, Премиум, Живые).
- **Внутренний рейтинг**: На основе логов (скорость старта, процент списаний, успешность выполнения) система присваивает услуге рейтинг. Если услуга провайдера начинает "боить", она автоматически заменяется на аналогичную или отключается.

## 3. Модуль "Provider Watchdog" (Мониторинг провайдера)
**Бизнес-логика**: Логирование каждого изменения в API VexBoost:
- Изменение цены (авто-пересчет нашей маржи).
- Изменение описания или лимитов (min/max).
- Включение/выключение услуги у провайдера.

## 4. Финансовый модуль
**Бизнес-логика**:
- Основная система: **ЮKassa** (для РФ).
- Резервная/Альтернативная: **Робокасса**.
- Валюта: RUB (с возможностью расширения).
