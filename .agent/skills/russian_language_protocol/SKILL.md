# Russian Language Protocol (RM-01)

## 📌 Описание
Данный протокол устанавливает обязательное использование русского языка для всех технических планов, документации, комментариев к коммитам (если не указано иное) и общения с пользователем в рамках проекта Smmplan.

## 🛠 Правила
1.  **Планы реализации (Implementation Plans)**: Всегда составляются на русском языке.
2.  **Артефакты (Artifacts)**: `task.md`, `walkthrough.md`, `implementation_plan.md` и любые другие отчеты должны быть на русском.
3.  **Общение**: Все ответы пользователю и системные уведомления (`notify_user`) должны быть на русском.
4.  **Технические термины**: Допускается использование английских терминов (названия функций, переменных, технологий), но описание их работы и обоснование выбора должны быть на русском.

## 🚀 Применение
Протокол применяется автоматически для всех последующих задач без напоминания.
