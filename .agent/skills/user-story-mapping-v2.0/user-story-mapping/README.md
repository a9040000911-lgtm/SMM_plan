## Установка

### Вариант 1: Локальная установка

```bash
# Создайте директорию для навыков (если не существует)
mkdir -p ~/.local/share/ai-assistant/skills

# Скопируйте директорию навыка
cp -r user-story-mapping ~/.local/share/ai-assistant/skills/
```

### Вариант 2: Установка в проект

```bash
# Скопируйте в директорию вашего проекта
cp -r user-story-mapping /path/to/your/project/.skills/
```

### Вариант 3: Использование с Claude/GPT

Скопируйте содержимое `SKILL.md` в системный промпт вашего AI-ассистента.

## Использование

### Базовый промпт

```
Создай карту пользовательских историй для [продукт].
Целевые пользователи: [описание].
Основные функции: [список].
```

### Примеры использования

**Новый продукт:**
```
Создай карту пользовательских историй для мобильного приложения 
доставки еды. Целевые пользователи — занятые профессионалы,
которые хотят быстро заказать еду.
```

**Существующий продукт:**
```
Создай карту историй для добавления функции комментариев 
в наш проектный менеджер. Пользователи — удалённые команды.
```

**Организация бэклога:**
```
Помоги организовать этот бэклог в карту историй: [вставить истории]
```

## Компоненты навыка

### 7-шаговый процесс

1. **Определение персон** — Кто пользователи?
2. **Каркас (Backbone)** — Основные активности
3. **Генерация историй** — Формат "As a... I want to... So that..."
4. **Приоритизация** — Must/Should/Could/Won't
5. **Разрезание релизов** — MVP и будущие версии
6. **Поиск пробелов** — Что упущено?
7. **Валидация** — Проверка с командой

### Формат вывода

Навык генерирует структурированный вывод:
- Персонажи и их цели
- Таблица историй по активностям
- MVP-срез
- Планы релизов
- Пробелы и вопросы

## Советы по фасилитации

### Подготовка к воркшопу

- Выделите 2-4 часа
- Подготовьте стикеры разных цветов
- Пригласите кросс-функциональную команду
- Используйте Miro/FigJam для удалённых команд

### Тайминг воркшопа

| Этап | Время |
|------|-------|
| Обзор персон | 10 мин |
| Генерация активностей | 15 мин |
| Упорядочивание | 10 мин |
| Добавление шагов | 20 мин |
| Написание историй | 30 мин |
| Приоритизация | 20 мин |
| Разрезание релизов | 15 мин |
| Поиск пробелов | 15 мин |
| Action items | 10 мин |
