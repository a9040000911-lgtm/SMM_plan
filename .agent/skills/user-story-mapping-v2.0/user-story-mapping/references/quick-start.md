# Quick-Start Template

Use this template to create a story map in 30 minutes.

## 5-Minute Context

```
SEGMENT: [Who? Be specific]
Example: "Busy parents of children aged 2-8"

PERSONA: [Name, age, role, pains, goals]
Example: "Maria, 34, working mom of two, struggles with meal planning,
         wants healthy meals without spending hours planning"

NARRATIVE: [One sentence goal]
Example: "Plan and prepare a week of healthy family dinners"
```

---

## 10-Minute Backbone

List 3-5 activities in order:

```
Activity 1: [What user does first]
Activity 2: [What user does second]
Activity 3: [What user does third]
Activity 4: [What user does fourth]
Activity 5: [What user does last] (optional)
```

**Quality check:**
- [ ] Activities are sequential
- [ ] Activities describe user actions, not features
- [ ] Activities are 3-5 in total

---

## 10-Minute Steps

For each activity, list 3-5 steps:

```
ACTIVITY 1: [Name]
  Step 1.1:
  Step 1.2:
  Step 1.3:

ACTIVITY 2: [Name]
  Step 2.1:
  Step 2.2:
  Step 2.3:

[Continue for all activities...]
```

**Quality check:**
- [ ] Each step is actionable
- [ ] Each step is observable
- [ ] Steps follow logical order

---

## 5-Minute Prioritization

For each step, mark tasks as:
- **MVP** = Essential for first release
- **R2** = Important but not critical
- **Future** = Nice to have

```
Step 1.1:
  - [Task] (MVP)
  - [Task] (MVP)
  - [Task] (R2)

Step 1.2:
  - [Task] (MVP)
  - [Task] (R2)
  - [Task] (Future)
```

## Tips for Speed

1. **Don't overthink activities** — Start with obvious ones, refine later
2. **Use your persona** — Ask "what would [persona name] do next?"
3. **Focus on flow** — Left-to-right is time, not importance
4. **MVP = 20-30%** — If everything is MVP, nothing is
5. **Come back later** — First draft in 30 min, refine with team
