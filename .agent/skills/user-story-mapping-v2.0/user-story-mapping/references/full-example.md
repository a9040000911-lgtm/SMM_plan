# Full Example: E-commerce Story Map

This document provides a complete story map example for an e-commerce platform.

## Story Map: Online Bookstore

### Context

**Segment:** 
Urban book readers aged 20-45 who purchase 5+ books per year, prefer online shopping for convenience and selection breadth.

**Persona:**
```
Name: Elena, 28, Marketing Manager
Location: Metropolitan area
Behavior: Reads 2-3 books per month, discovers books via social media
          and recommendations, values fast delivery
Pain Points: 
  - Hard to find specific editions
  - Shipping costs add up
  - Returns are inconvenient
Goals: 
  - Discover interesting books easily
  - Get books quickly without high shipping
  - Return books hassle-free if needed
```

**Narrative:**
Find and purchase an interesting book with confidence it will arrive quickly and can be returned if needed.

---

### Backbone (Activities)

```
1. DISCOVER → 2. EVALUATE → 3. PURCHASE → 4. RECEIVE → 5. POST-PURCHASE
```

---

### Detailed Map

#### Activity 1: DISCOVER

| Step | MVP Tasks | Release 2 | Future |
|------|-----------|-----------|--------|
| **1.1 Browse catalog** | View featured books, Search by title/author, See book covers | Genre categories, Bestsellers list | AI recommendations, "Books like this" |
| **1.2 Get recommendations** | Show related books, Display "Others bought" | Personalized picks | Friend recommendations, Book clubs |
| **1.3 Check availability** | Show in-stock status, Display formats available | Pre-order option, Back-in-stock alert | Multi-store inventory |

#### Activity 2: EVALUATE

| Step | MVP Tasks | Release 2 | Future |
|------|-----------|-----------|--------|
| **2.1 Read details** | View book description, See page count, Check publisher info | Sample chapters, Table of contents | Age rating, Content warnings |
| **2.2 Check reviews** | Display star rating, Show review count | Read reviews, Filter by rating | Verified purchase badge, Helpful votes |
| **2.3 Compare options** | See price per format, View condition (new/used) | Compare sellers, Price history | Wishlist comparison |
| **2.4 Verify decision** | Add to cart, Save for later | Share to social, Ask a friend | Gift finder, Reading level |

#### Activity 3: PURCHASE

| Step | MVP Tasks | Release 2 | Future |
|------|-----------|-----------|--------|
| **3.1 Review cart** | See item list, View total price, Adjust quantities | Apply coupon, See savings | Cart reminders, Saved for later |
| **3.2 Enter shipping** | Input address, Select speed | Saved addresses, Address validation | Multiple addresses, Gift wrapping |
| **3.3 Choose payment** | Credit card input, PayPal | Saved cards, Apple/Google Pay | Buy now pay later, Store credit |
| **3.4 Confirm order** | Review summary, Place order | Edit before submit, Estimated delivery | One-click reorder |

#### Activity 4: RECEIVE

| Step | MVP Tasks | Release 2 | Future |
|------|-----------|-----------|--------|
| **4.1 Track shipment** | Order confirmation email, Shipping notification | Tracking number, Carrier link | Real-time map, Delivery window |
| **4.2 Get delivery** | Basic packaging, Packing slip | Gift message, Gift receipt | Photo on delivery, Safe place options |
| **4.3 Unpack** | Item received, Condition check | Care instructions, Bookmark included | Unboxing experience, Bonus content |

#### Activity 5: POST-PURCHASE

| Step | MVP Tasks | Release 2 | Future |
|------|-----------|-----------|--------|
| **5.1 Evaluate** | Rate purchase, Write review | Photo reviews, Video reviews | Reading progress, Finished date |
| **5.2 Return if needed** | Return policy visible, Return request form | Prepaid label, Pickup scheduling | Instant refund, Exchange option |
| **5.3 Engage further** | Newsletter signup | Loyalty points, Member benefits | Book clubs, Author events |

---

### Release Planning

**MVP (Release 1) — Core Purchase Flow:**
- Browse: Search, featured books, basic categories
- Evaluate: Description, ratings, price comparison
- Purchase: Cart, checkout, basic payment
- Receive: Confirmation, shipping notification
- Post: Basic returns, simple reviews

*Target: 8-10 weeks development*

**Release 2 — Enhanced Experience:**
- Personalized recommendations
- Saved addresses and payment methods
- Detailed reviews with photos
- Loyalty program basics
- Better tracking

*Target: 4-6 weeks after MVP*

**Future — Differentiation:**
- AI-powered recommendations
- Book clubs integration
- Subscription options
- Author events
- Advanced returns

---

### Gaps & Opportunities

**Identified Gaps:**
1. No international shipping handling
2. Missing gift card purchase flow
3. No digital/ebook option mentioned
4. Affiliate/influencer discovery path unclear

**User Pain Points:**
1. Shipping costs not shown until checkout
2. Can't see inside the book before buying
3. Returns require printing labels
4. No way to try audiobook sample

**Delight Opportunities:**
1. "Blind date with a book" mystery selection
2. Personalized book notes from staff
3. Reading challenges with rewards
4. Local book club connections

---

### Key Metrics to Track

| Metric | MVP Target | Release 2 Target |
|--------|------------|------------------|
| Cart abandonment | < 70% | < 50% |
| Search to purchase | > 5% | > 8% |
| Return rate | < 10% | < 7% |
| Review rate | > 3% | > 10% |
| Repeat purchase (90 days) | > 15% | > 25% |

## How to Use This Example

1. **Adapt the structure** for your specific product
2. **Customize activities** based on your user's actual journey
3. **Adjust tasks** to match your technical constraints
4. **Re-prioritize** based on your business goals
5. **Add/remove activities** as needed (keep 3-5)
