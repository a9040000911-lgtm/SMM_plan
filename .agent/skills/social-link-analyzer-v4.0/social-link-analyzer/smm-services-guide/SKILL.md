# SMM Services Guide

Полный справочник по соответствию SMM-услуг и типов ссылок для социальных сетей.

## Как использовать этот навык

Используйте этот навык, когда:
- Клиент заказывает SMM-услугу и нужно определить правильный формат ссылки
- Нужно проверить соответствие типа ссылки запрашиваемой услуге
- Требуется объяснить клиенту, какую ссылку предоставить

## VK (ВКОНТАКТЕ)

### Подписчики

| Услуга | Формат ссылки | Пример | Примечание |
|--------|---------------|--------|------------|
| Подписчики на пользователя | `vk.com/id123` или `vk.com/username` | `vk.com/durov` | Профиль должен быть открыт |
| Подписчики на группу | `vk.com/club123` или `vk.com/groupname` | `vk.com/aprilclub` | Открытая группа |
| Подписчики на паблик | `vk.com/public123` или `vk.com/pagename` | `vk.com/public123456` | — |
| Подписчики на мероприятие | `vk.com/event123` | `vk.com/event123456` | — |

### Лайки

| Услуга | Формат ссылки | Пример |
|--------|---------------|--------|
| Лайки на пост | `vk.com/wall-123_456` | `vk.com/wall-22884714_12345` |
| Лайки на фото | `vk.com/photo-123_456` | `vk.com/photo-22884714_457654321` |
| Лайки на видео | `vk.com/video-123_456` | `vk.com/video-22884714_456239687` |
| Лайки на клип | `vk.com/clip-123_456` | `vk.com/clip-22884714_456239487` |
| Лайки на товар | `vk.com/product-123_456` | — |

### Просмотры

| Услуга | Формат ссылки | Пример |
|--------|---------------|--------|
| Просмотры видео | `vk.com/video-123_456` | `vk.com/video-22884714_456239687` |
| Просмотры клипа | `vk.com/clip-123_456` | `vk.com/clip-22884714_456239487` |
| Просмотры stories | `vk.com/username` (ссылка на профиль) | — |

## Сводная таблица ошибок

| Платформа | Ошибка | Правильный формат |
|-----------|--------|-------------------|
| VK | `m.vk.com/...` | `vk.com/...` |
| VK | `wall123_456` (без минуса для группы) | `wall-123_456` |
| Instagram | `instagram.com/username` для лайков | `instagram.com/p/XXXXXXX/` |
| Instagram | `/p/...` для Reels | `/reel/...` |
| TikTok | `vm.tiktok.com/...` | `tiktok.com/@username/video/...` |
| TikTok | Ссылка на профиль для просмотров видео | Ссылка на видео |
| YouTube | `youtube.com/watch` без `?v=` | `youtube.com/watch?v=VIDEO_ID` |
