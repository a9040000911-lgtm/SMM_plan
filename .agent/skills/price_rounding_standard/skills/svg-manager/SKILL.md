# SVG Manager Skill

Этот скилл позволяет агенту находить официальные SVG-иконки брендов и добавлять их в проект.

## Возможности
1. **Загрузка файла**: Скачивает SVG файл в папку `src/assets/icons/`.
2. **Инлайнинг**: Может прочитать содержимое SVG и вставить его прямо в HTML/React компонент.

## Как использовать
- Когда пользователь просит "Добавь иконку [Название]", используй скрипт `fetch_svg.py`.
- Всегда уточняй путь сохранения: по умолчанию `src/assets/icons/`.
- Если проект на React, предлагай создать отдельный компонент для иконки.

## Источник данных
Используется CDN Simple Icons (simpleicons.org).
