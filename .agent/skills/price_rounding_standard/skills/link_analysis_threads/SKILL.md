# Threads Link Analysis Skill (v1.0 - The Meta Thread)

Разбор архитектуры ссылок Threads, глубокой интеграции с Instagram и алгоритмов Meta по борьбе с фродом в 2026 году.

## 1. Анатомия и паттерны ссылок

Threads тесно связан с инфраструктурой Instagram (Project Barcelona), но использует свой домен.

### Основные форматы (Regex Pattern):
1. **Web Links (Standard)**:
   - Формат: `threads.net/@username/post/postid`
   - **Regex**: `threads\.net\/@([a-zA-Z0-9._-]+)\/post\/([a-zA-Z0-9_-]+)`

2. **Mobile Deep Links (Barcelona Protocol)**:
   - Формат: `barcelona://user?username={username}`
   - Используется для мгновенного открытия приложения.
   - **Regex**: `barcelona:\/\/user\?username=([a-zA-Z0-9._-]+)`

3. **Share Intents**:
   - Формат: `threads.net/intent/post?text=...`

## 2. Интеграция с Instagram (The Shared Root)

В 2026 году Threads перестал быть "отдельным сервисом" и стал расширением Instagram.

- **Cross-posting**: Посты из Threads автоматически летят в Instagram Stories и ленту (и наоборот). 
- **Follower Sync**: При накрутке в Threads важно, чтобы у ботов были **связанные Instagram-аккаунты**. Если на Threads-аккаунт подписано 1000 ботов, у которых нет профилей в Instagram, Meta помечает эту активность как "Изолированный спам" и обнуляет счетчик.

## 3. Алгоритмы и Антифрод 2026

Threads использует комбинированный ИИ Meta (Llama 4-based) для анализа контента.

### A. Real-time Topic Discovery (Алгоритм "Первого часа")
- Алгоритм Threads 2026 вознаграждает вовлеченность **в первый час** после публикации. 
- Накрутка лайков и репостов должна быть максимально активной сразу после выхода поста, чтобы вывести его в раздел "Trending Topics".

### B. Facial Recognition & Account Recovery
Meta активно тестирует распознавание лиц для борьбы с "celebrity impersonation" (дипфейками знаменитостей). 
- **Риск для SMM**: Аккаунты ботов с лицами реальных людей/селебрити могут быть заблокированы автоматически при первой же проверке нейросетью.

### C. Fediverse Integration (Decentralized Discovery)
Threads интегрирован в ActivityPub. Это значит, что пост может распространяться за пределы Meta. 
- **Возможность**: "Накрутка" через сторонние Mastodon-серверы может выглядеть как органический внешний трафик для алгоритмов Threads.

## 4. Алгоритм для SMM-панели (Best Practice)
1. **IG-Link Verification**: Проверка наличия связанного Instagram-профиля у аккаунта клиента. Без него накрутка малоэффективна.
2. **Intent Cleaning**: Очистка ссылок от параметров отслеживания после репостов из Instagram.
3. **Engagement Mix**: Соотношение "Лайк : Репост : Ответ" должно быть в рамках тематического тренда. В Threads "Ответы" (Replies) имеют самый высокий вес для попадания в рекомендации.
4. **Visual Boost**: Текстовые посты с картинками/видео получают в 3 раза больше органического охвата. Рекомендовать клиентам добавлять медиа-контент.
