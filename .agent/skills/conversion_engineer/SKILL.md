# Conversion Engineer Skill (v1.0)

Этот скилл управляет математической и психологической эффективностью интерфейса Smmplan.

## 1. Фундаментальные Benchmarks 2026
*   **Правило 3-х секунд**: Пользователь должен понять УТП и увидеть основной CTA в течение 3 секунд.
*   **Single Focus**: Одна страница — одна главная задача (Conversion Goal). Минимум отвлекающих факторов (убрать лишнюю навигацию в Hero).
*   **Mobile-First Speed**: Приоритет скорости загрузки и «thumb-friendly» зон.

## 2. Нейромаркетинг и Психология
*   **Cognitive Ease**: Использовать знакомые паттерны (F-shape для текста, Z-shape для лендинга).
*   **Цветовая стимуляция**:
    *   `Indigo/Blue`: Постоянный фон (Доверие).
    *   `Orange/Amber`: Акценты на выгоде (Энергия).
    *   `Red (P3)`: Критические CTA кнопки (Срочность).
*   **Trust Signals**: Располагать отзывы и счетчики (1.2M+ заказов) максимально близко к полю ввода ссылки.

## 3. CRO-Чек-лист для SMM-панелей
- [ ] Отражен ли «Конечный результат» в заголовке (а не процесс)?
- [ ] Виден ли главный CTA без скролла?
- [ ] Есть ли «AI Guarantee» индикатор рядом с формой заказа?
- [ ] Сокращено ли количество шагов до оформления до 1 (One-click flow)?
