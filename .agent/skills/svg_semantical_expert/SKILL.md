# SVG & Visual Semantics Protocol

Этот скилл управляет выбором и использованием иконок для усиления маркетингового сообщения.

## 1. Правило Семантической Связи (Icon-Text Grip)
Иконка не должна быть просто «красивой». Она должна подтверждать текст:
*   **Безопасность** -> Shield (не замок, если речь о защите данных).
*   **Скорость** -> Zap/Rocket (не часы, которые ассоциируются с ожиданием).
*   **AI/Интеллект** -> Sparkles/Brain (не просто шестеренка).

## 2. Техническая Вставка (SVG Guard)
- Всегда использовать `lucide-react` как базу.
- Если нужна кастомная иконка: использовать чистый SVG с `fill="currentColor"`.
- Масштабируемость: четкие `size` пропсы (16/18/20/24).

## 3. Визуальный Ритм
- Иконки в одном ряду должны иметь одинаковую толщину линий (stroke-width).
- Использовать `opacity` для иконок-акцентов, чтобы они не отвлекали от главного CTA.
