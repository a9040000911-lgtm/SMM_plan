# UI Quality Guard Protocol (v1.0)

Этот скилл предотвращает «грязную» верстку и визуальные баги.

## 1. Проверка Наложения (Overlay Audit)
Прежде чем использовать `fixed`, `absolute` или `z-index`, ответь на вопросы:
- Не перекроет ли этот элемент навигационное меню?
- Виден ли этот элемент на мобильных устройствах (Viewport check)?
- Есть ли `backdrop-blur`, если элемент накладывается на другой текст?

## 2. Layout Math (Математика отступов)
- Использовать только шаг 4/8/16 пикселей.
- Проверять "Визуальный шум": если в блоке более 3-х разных размеров шрифта — это ошибка.
- Центрирование: проверять реальный центр элемента через `browser_subagent`.

## 3. Состояния перехода (The "Between" Checks)
Каждый интерактивный элемент должен иметь:
- `hover`: Плавный переход (0.2s - 0.3s).
- `focus`: Четкий контур для доступности.
- `active`: Микро-масштабирование (scale-95).

## 4. Алгоритм «Поиск Артефактов»
1. Открой страницу в браузере.
2. Скролль до низа и обратно.
3. Если во время скролла элементы «дергаются» или накладываются некорректно — вернись к коду.
4. Проверь "Сломанные изображения" и плейсхолдеры.
