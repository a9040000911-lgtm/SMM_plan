# Дорожная карта навыков ИИ для создания безупречного интерфейса (AI Interface Excellence)

Этот документ определяет направления развития «интеллекта дизайнера» внутри ИИ для достижения уровня Visionary 2.0.

## 1. Навыки Семантики и Лингвистики (Semantic Intelligence)
*   **PAS/AIDA Master**: Искусство упаковывать пользу в маркетинговые фреймворки (Problem-Agitation-Solution).
*   **Microcopy Harmony**: Подбор слов для кнопок, тултипов и плейсхолдеров, которые снижают когнитивную нагрузку.
*   **Headline Magnetism**: Создание заголовков, которые отвечают на "Jobs to Be Done" пользователя.
*   **Tone of Voice Adaptation**: Способность мгновенно переключаться между "Premium Tech" и "Friendly Support" в зависимости от блока.

## 2. Навыки «Слепого проектирования» (Headless Design)
*   **Layout Math**: Проектирование сеток через математические ограничения (Constraints) и коэффициенты, а не визуальное «на глаз».
*   **Atomic Tokenization**: Умение работать исключительно с дизайн-токенами (spacing-4, radius-2xl, surface-glass), создавая системную консистентность.
*   **Symbolic Positioning**: Определение иерархии через древовидные структуры (DOM-logic), понимая веса элементов без их отрисовки.

## 3. Навыки Визуальной Логики и Геометрии (Geometric Logic)
*   **Optical Balance**: Понимание визуального центра тяжести и балансировка асимметричных макетов.
*   **Mathematical Spacing (8px System)**: Жесткое следование кратности 4/8/16 для создания ритма интерфейса.
*   **Depth Hierarchy**: Управление слоями (Z-index) и тенями для создания «Карты глубины» (Liquid Glass).

## 4. Навыки Нейромаркетинга и CRO (Behavioral Intelligence)
*   **Friction Detection**: Поиск и устранение лишних полей, кликов и текстов, мешающих конверсии.
*   **Trust Anchoring**: Стратегическое размещение Social Proof (отзывы, счетчики) в «зонах горячего внимания».
*   **Chameleon Adaptation**: Логика динамической смены тем под контекст пользователя (Platform Detection).

## 5. Техническое Совершенство (Code-as-Design)
*   **Performance-Driven UI**: Создание дизайнов, которые эффективно рендерятся и не перегружают DOM.
*   **Accessibility Guard (WCAG)**: Автоматическое соблюдение контрастности и семантики HTML для скринридеров.
*   **Responsive Hybridism**: Проектирование переходов между Mobile и Desktop через адаптивные хуки и контейнерные запросы.
