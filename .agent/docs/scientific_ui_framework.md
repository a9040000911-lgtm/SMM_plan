# Smmplan Scientific UI/UX Framework (v2.0)

Этот документ является «Золотым стандартом» для проектирования интерфейсов Smmplan, объединяя науку о данных, психологию и визуальное искусство.

## 1. Триада Эффективности
1.  **Visuality (Visionary UI Engineer)**: Живое стекло, P3-цвета, микро-анимации, адаптивность.
2.  **Conversion (Conversion Engineer)**: Single CTA, Нейромаркетинг, Безопасность (Trust Signals).
3.  **Semantics (Semantic Auditor)**: Логическая чистота, Magnetic Headlines, Harmony.

## 2. Формула Идеального Hero-блока
*   **Заголовок**: [Сильный глагол] + [Конечный результат] + [Эмоция/Статус]. 
    *   *Пример*: «Масштабируйте влияние мгновенно. Профессиональный рост без границ.»
*   **Визуальный акцент**: AI Hub (Интерактивная точка входа). Должен занимать >40% площади Hero.
*   **Trust Buffer**: Счетчики (Social Proof) должны быть в радиусе 100px от точки ввода ссылки.

## 3. Цветовой Код Конверсии
| Элемент | Цвет (P3 / HEX) | Психологический эффект |
| :--- | :--- | :--- |
| **Фон (Glass)** | `Slate-950 (Opacity 20-40%)` | Глубина, профессионализм |
| **Акцент (Action)** | `Blue-600 / Indigo-500` | Доверие и технологичность |
| **Выгода (Highlight)** | `Amber-400 / Orange-500` | Энергия и вознаграждение |
| **Критический CTA** | `Crimson-600 / Red-500` | Срочность (использовать редко) |

## 4. Чек-лист научного аудита (Score 0-10)
1.  **Clarity (0-2)**: Понятно ли за 3 сек, что делать?
2.  **Trust (0-2)**: Есть ли доказательства надежности на первом экране?
3.  **Logic (0-2)**: Есть ли в текстах бесполезные слова?
4.  **Flow (0-2)**: Сколько кликов до заказа? (Идеально — 1).
5.  **Aesthetics (0-2)**: Вызывает ли дизайн чувство «Премиальности»?

**Минимальный проходной балл: 8/10.**
